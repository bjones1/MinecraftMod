=====================
Minecraft farming mod
=====================
This mod adds a set of farming-related things to Minecraft.

Installing
==========
To install this mod, follow `these directions <http://www.minecraftmods.com/how-to-install-mods-for-minecraft-forge/>`_.

If you'd like to modify this mod, first set up a Minecraft development environment by following the directions in `README.txt <https://github.com/MinecraftForge/MinecraftForge/blob/master/README.txt>`_.
